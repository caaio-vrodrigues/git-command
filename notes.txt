Novo repositorio:
git remote add origin https://github.com/caaiorr/teste-github

Forçar push:
git push -f origin main

Histórico:
git reflog

Voltar versão:
git reflog (pegar código versão)
git reset --hard (colar código versão)

Lista de branchs:
git branch

Nova branch:
git checkout -b (nova branch) main

Mudar de branch:
git checkout (nome branch)

Merge:
git branch main
git pull
git merge (nome branch)
git push
